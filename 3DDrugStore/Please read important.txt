To start the system:
Create an ODBC
- open control panel
- open administration tools if you are using XP, or find ODBC
- open ODBC
- click System DSN, then Click Add
- From the list of driver select (Microsoft Access Driver (*.mdb) thin click finish
- On the Data Source Name type POS
- Click Select Button,then find the POS database, that is stored in the database folder of the system.
- Press ok, then you can start the system.
